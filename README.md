# Java Hands-On Lab Programs

This repository contains my Java hands-on lab programs developed during my practical coursework.  
It includes implementations of core Java concepts, problem-solving exercises, and basic applications written in the lab.

---

## Topics Covered

- Basic syntax and structure  
- Data types, variables, and operators  
- Conditional statements (if, else, switch)  
- Looping constructs (for, while, do-while)  
- Arrays (1D and 2D)  
- Strings and string functions  
- Methods and recursion  
- Object-Oriented Programming  
  - Classes and objects  
  - Constructors  
  - Inheritance  
  - Polymorphism  
  - Abstraction  
  - Encapsulation  
- Exception handling  
- File handling  
- Basic problem-solving programs  

---

## How to Run the Programs

1. Install JDK 8 or above.  
2. Open any `.java` file in your preferred IDE (IntelliJ, Eclipse, VS Code).  
3. Compile the program:
   ```bash
   javac FileName.java
4. Run the program:
java FileName
